# Dataset

Current dataset I plan on using for the final training run: [Spotify's million song dataset](https://www.aicrowd.com/challenges/spotify-million-playlist-dataset-challenge/dataset_files)

For training, I plan on using a smaller version of this dataset.

`process_raw_dataset.py` takes the raw spotify dataset and creates two other datasets from it. A cleaned up playlist -> songs dataset and an inverted songs -> playlists dataset.
